# 💍 K-kun & Haru's Wedding Project

這是一個專為我們婚禮打造的互動式網頁，旨在提供賓客最流暢的資訊查詢體驗。

## 🌸 專案亮點

- **森林系視覺設計**：採用大地色系 (#454837, #f3efeb) 營造溫潤質感。
- **響應式排版**：使用 React-Bootstrap 確保手機與電腦瀏覽體驗一致。
- **動態效果**：結合 AOS (Animate On Scroll) 增加頁面捲動的優雅節奏感。
- **整合 Google Maps**：提供精確的交通資訊與導航功能。

## 🛠 使用技術

- **前端框架**: React (Vite)
- **樣式管理**: Sass (SCSS) / Bootstrap 5
- **動畫庫**: AOS
- **部署平台**: GitHub Pages (預計未來遷移至 AWS)
